# part2-rfm-retention-strategy
## Key Findings

- Dormant customers exhibited the highest churn rate (90.1%).
- Champions exhibited the lowest churn rate (10.5%).
- High Value Unhappy customers generated substantial revenue but showed elevated churn risk (72.3%).
- Discount Sensitive customers displayed strong dependence on promotional incentives.
- RFM combined with behavioral signals provided significantly more actionable segmentation than RFM alone.

## Business Recommendation

Retention budget should prioritize:

1. High Value Unhappy
2. At Risk
3. Discount Sensitive

These segments represent the highest revenue recovery opportunity.
